# Bike-Rental-System
A console-based Bike Rental System developed using Java, JDBC, and MySQL.
The project supports Admin and Customer roles with proper database interaction.
# Features
  # Admin
 Login authentication.
 
 Add, delete, and view bikes.
 
 View all rented bikes.
 
 Logout without closing the app.

# Customer
  Registration & login.
  
  View available bikes.
  
  Rent multiple bikes.
 
  View rented bikes.
  
  Return bikes.
  
  Logout to previous menu.
